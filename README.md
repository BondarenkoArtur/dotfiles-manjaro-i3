# .dotfiles for manjaro-i3

## Setup and installation
These dotfiles are built using [dotbot](https://github.com/anishathalye/dotbot)
```
$ git clone git@github.com:francoiscote/dotfiles-manjaro-i3.git .dotfiles
$ cd .dotfiles
$ chmod +x install
$ ./install
```
